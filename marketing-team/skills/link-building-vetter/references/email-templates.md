# Email Response Templates

Tone: Polite, constructive, strict with guidelines. Works for email or LinkedIn.

## Approval (Score 8-10)

```
Hi [Name],

Thanks for sending this over. I've reviewed your proposal and it looks good to go.

[If any minor notes, add here]

I'll get this live shortly.

Best,
Stefan
```

## Revision Request (Score 5-7)

```
Hi [Name],

Thanks for the proposal. I've reviewed it against our guidelines and have some feedback:

[Specific issue 1]
[Specific issue 2]

To move forward, please:
[Clear action 1]
[Clear action 2]

Happy to proceed once these are addressed.

Best,
Stefan
```

### Common Revision Requests

**Anchor too long or competitive:**
```
The anchor "[anchor]" is a bit too keyword-heavy for us. Could you shorten it or use something more natural? We're looking for anchors that read naturally in context rather than targeting specific search terms.
```

**Placement in intro/conclusion:**
```
We don't place links in introductions or conclusions. Could you suggest a placement in the body of the article where it fits more naturally with the surrounding content?
```

**Text doesn't match tone:**
```
The proposed text doesn't quite match the article's tone. This piece is [describe tone - casual/formal/technical]. Could you revise to better match that style?
```

**AI-sounding phrases:**
```
The text reads a bit AI-generated, particularly phrases like "[specific phrase]". We need 100% human-written content that matches our writing style. Could you revise?
```

**Off-topic modification:**
```
The proposed text doesn't align with this section's topic. This paragraph discusses [topic], so the addition needs to connect directly to that. Could you propose something more contextually relevant?
```

**Target page competes with article:**
```
The target page appears to compete with our article's keywords. We can't link to content that targets the same terms we're ranking for. Do you have an alternative page that's complementary rather than competitive?
```

## Rejection (Score 1-4)

```
Hi [Name],

Thanks for reaching out. Unfortunately, this one doesn't work for us:

[Primary reason]

This is a firm guideline on our end, so I won't be able to make exceptions here.

If you have other opportunities that fit our guidelines, I'm happy to take a look.

Best,
Stefan
```

### Common Rejection Reasons

**Excluded article:**
```
This article isn't available for link exchanges. It's on our excluded list due to [performance/strategic reasons].
```

**Restricted category:**
```
We don't accept links from [category] sites. This is a firm policy.
```

**DR too low:**
```
The target domain's DR is below our 50 minimum threshold. We need DR 50+ for all link exchanges.
```

**Wrong page type:**
```
We only link to blogs, articles, and glossaries. [Homepages/product pages/landing pages/service pages] don't qualify.
```

**Location-specific or niche page:**
```
We don't link to location-specific pages or highly niche industry content. These don't align with our blog's broad audience.
```

## Follow-Up (No Response)

```
Hi [Name],

Just following up on the link exchange proposal from [date]. Let me know if you'd like to proceed or if you have any questions about our feedback.

Best,
Stefan
```
